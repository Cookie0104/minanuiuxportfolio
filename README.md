# Min-An's Portfolio

個人作品集網站，展示 UI/UX 設計作品與專案經驗。

## 功能特色

- 📱 響應式設計，支援多種裝置
- 🌐 多語言支援（中文/英文）
- 🎨 文青風格設計
- 🖼️ 作品展間（Gallery）
- 📝 留言互動功能
- 🎨 靈感實驗室（Canvas 繪圖功能）

## 頁面結構

- **首頁** (`index.html`) - 個人介紹與靈感實驗室
- **關於我** (`about.html`) - 個人簡介、技能與工作經歷
- **作品集** (`works.html`) - 專案介紹
- **作品展間** (`gallery.html`) - 設計作品展示
- **留言互動** (`contact.html`) - 訪客留言功能

## 技術棧

- HTML5
- CSS3 (使用 CSS Variables 設計系統)
- Vanilla JavaScript
- Canvas API (繪圖功能)

## 本地運行

1. 下載或克隆此專案
2. 直接用瀏覽器開啟 `index.html` 即可瀏覽
